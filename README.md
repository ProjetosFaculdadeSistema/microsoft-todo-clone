# TaskMaster — Clone do Microsoft To Do

Aplicação feita em Angular que simula o Microsoft To Do, com autenticação local, gerenciamento de tarefas e layout moderno com TailwindCSS.

## Tecnologias Utilizadas

- Angular 17
- TailwindCSS
- TypeScript
- Git e GitHub
- localStorage (simulação de backend)

## Funcionalidades

- Registro e login de usuários
- Proteção de rotas com guard
- Persistência de sessão
- Adicionar, concluir, filtrar e excluir tarefas
- Layout responsivo inspirado no Microsoft To Do

## Como rodar o projeto

1. Clonar o repositório

git clone https://github.com/ProjetosFaculdadeSistema/microsoft-todo-clone.git

2. Acessar a pasta

cd microsoft-todo-clone

3. Instalar as dependências

npm install

4. Rodar o projeto

ng serve

5. Acessar no navegador:

http://localhost:4200
